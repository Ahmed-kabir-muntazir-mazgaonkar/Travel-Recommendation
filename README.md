## Features

- Personalized travel recommendations based on user preferences.
- Interactive and user-friendly interface.
- Lightweight and fast; runs locally or can be deployed on platforms like Hugging Face Spaces.

---

## Technologies Used

- **Python 3**
- **Streamlit**
- **Pandas**
- **Scikit-learn**
